# Đại hội Hội LHTN Việt Nam Phường Xuân Hương Lần I

Dự án này là trang web (Landing Page) chính thức phục vụ cho **Đại hội đại biểu Hội Liên hiệp Thanh niên Việt Nam phường Xuân Hương – Đà Lạt lần thứ I, nhiệm kỳ 2026 - 2029**. 

Trang web được thiết kế hiện đại, trẻ trung, và chuyên nghiệp nhằm cung cấp thông tin, tài liệu đại hội, cũng như tạo không gian tương tác trực tuyến cho các đại biểu và đoàn viên thanh niên.

## 🌟 Tính năng nổi bật

- **Giao diện thân thiện, hiện đại (Hero Section):** Cung cấp thông tin trực quan về thời gian, địa điểm tổ chức đại hội kèm hiệu ứng bắt mắt.
- **Thống kê "Dấu ấn" (Stats):** Thể hiện những thành tựu, số liệu nổi bật của phong trào thanh niên giai đoạn 2024 - 2026 thông qua các thẻ thông tin sinh động.
- **Tài liệu Đại hội (Documents):** Tích hợp liên kết xem trực tuyến văn kiện đại hội dưới dạng sách lật (Flipbook).
- **Chỉ tiêu trọng tâm (Targets):** Trình bày rõ ràng các mục tiêu, sứ mệnh trong nhiệm kỳ mới (2026 - 2029) với các biểu tượng trực quan.
- **Thước phim Đại hội (Gallery):** Hiển thị hình ảnh hoạt động dưới dạng băng chuyền (marquee) tự động cuộn hai chiều đẹp mắt.
- **Thanh niên hiến kế (Feedback Form):** Biểu mẫu trực tuyến cho phép đại biểu đóng góp ý kiến, sáng kiến cho các phong trào và công tác hội.
- **Trình phát nhạc nền (Music Player):** Tích hợp trình phát nhạc bài hát "Lên Đàng" với tính năng hiển thị và chạy lời bài hát đồng bộ (Karaoke style).

## 🛠️ Công nghệ sử dụng

- **Cấu trúc & Giao diện:** HTML5, CSS3 (Vanilla CSS).
- **Ngôn ngữ kịch bản:** JavaScript (Vanilla JS).
- **Biểu tượng:** FontAwesome 6.4.0.
- **Phông chữ:** Các phông chữ hiện đại (Google Fonts).

## 📂 Cấu trúc thư mục

```text
DH_LHTN_VN/
├── index.html        # Trang chính của Landing Page
├── image/            # Thư mục chứa hình ảnh tĩnh, logo, ảnh gallery
├── video/            # Thư mục chứa tệp đa phương tiện (nhạc nền, video)
├── styles/           # Thư mục chứa các tệp CSS (main.css,...)
├── scripts/          # Thư mục chứa các tệp JavaScript xử lý logic
└── README.md         # Tài liệu mô tả dự án
```

## 🚀 Hướng dẫn chạy dự án

Dự án này sử dụng HTML/CSS/JS thuần tĩnh. Bạn không cần cài đặt môi trường phức tạp:

1. Clone hoặc tải mã nguồn về máy.
2. Mở tệp `index.html` bằng bất kỳ trình duyệt web nào (Chrome, Firefox, Edge, Safari...).
3. Để có trải nghiệm tốt nhất (đặc biệt với các tính năng như tải tài nguyên, fetch API nếu có sau này), khuyến khích chạy thông qua một Live Server (ví dụ: extension "Live Server" trên VS Code).
